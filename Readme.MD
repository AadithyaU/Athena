Athena was an online coding event by Kurukshetra '15, where participants have to code for 20 questions in a span of 2 hours. I was part of both organizer team and web team of this event.Website was built with "Mobile first" concept as the base. The site runs on phones, tablets and PCs with no issues. We used Twitter bootstrap.

The project code will be re-used next year and so linking the code is not possible. However pictures of the site are linked.
